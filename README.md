## :globe_with_meridians: URL

https://kyoruni.github.io/badge-generator/

## :wrench: セットアップ

```
$ git clone https://github.com/kyoruni/badge-generator.git
```

```
$ npm install
```

```
$ npm run serve
```

## :bee: ビルド

- docs ディレクトリにビルドされます

```
$ npm run build
```
